Redirecting to quillmark.dev
============================

.. raw:: html

   <script>
      var path = window.location.pathname.replace(/^\/(?:projects\/[^/]+\/)?[a-z]{2}(?:[-_][a-z]+)?\/[^/]+/i, "");
      if (!path.startsWith("/")) path = "/" + path;
      var query = window.location.search;
      var hash = window.location.hash;
      window.location.replace("https://quillmark.dev" + path + query + hash);
   </script>
   <meta http-equiv="refresh" content="0; url=https://quillmark.dev/" />

The documentation has moved to `quillmark.dev <https://quillmark.dev/>`_.
